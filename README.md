# 🌟 ExpressVPN Web - Fast & Secure VPN Simulator

**A sleek, fully functional browser-based VPN simulator that replicates the official ExpressVPN Android app (com.expressvpn.vpn).**

Connect instantly to 105 countries, experience the Lightway protocol, toggle Kill Switch & Split Tunneling, run speed tests, and enjoy a beautiful dark UI with real-time connection status — all running 100% locally in your browser.

## ✨ Features
- 🚀 **One-tap Connect / Disconnect** with beautiful animation  
- 🌍 **105 Country Server List** – clickable with flags & ping  
- ⚡ **Lightway Protocol Simulation** – fastest VPN protocol  
- 🛡️ **Kill Switch & Split Tunneling** – demo toggles in UI  
- 📊 **Live Speed Test** – realistic download/upload results  
- 🔒 **No-Logs Privacy Badge** + military-grade encryption visuals  
- 📱 **100% Mobile-First** – perfect on phones, tablets & desktop  
- 🌐 **SEO Optimized** – rich meta tags, keywords & semantic HTML  
- 🎨 **Premium Dark UI** with emerald green accents & smooth transitions  
- 🔄 **Real-time IP & Location Change** on connect
